# Deployment

This folder contains the deployment files for the IPL Match Winner Prediction application.

## Files

### app.py

Streamlit web application used to generate IPL match winner predictions.

### App SS.png

Screenshot of the deployed application's user interface.

### KKR vs PBKS Deployment.png

Sample prediction output generated by the deployed application.

## Application Features

* Select Team 1
* Select Team 2
* Select Venue
* Generate match winner prediction instantly

## Deployment Technology

* Python
* Streamlit
* Scikit-Learn
* Pickle

The application uses the trained machine learning model stored in the Model-Files folder.
