# Notebook

This folder contains the complete machine learning workflow used for the IPL Match Winner Prediction project.

## Files

### Model.ipynb

The notebook includes:

* Data Collection
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Encoding and Scaling
* Model Training
* Model Evaluation
* Feature Importance Analysis
* IPL 2026 Predictions

## Models Implemented

* Logistic Regression
* Random Forest
* XGBoost
* Extra Trees
* LightGBM
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Multi-Layer Perceptron (MLP)

The notebook serves as the complete end-to-end development pipeline for the project.
