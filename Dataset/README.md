# Dataset

This folder contains the datasets used for the IPL Match Winner Prediction project.

## Files

### ipl-2026-UTC.csv

Contains IPL 2026 fixture data used for generating future match winner predictions.

### Historical IPL Dataset

The complete historical IPL dataset (2008–2025) used for model training exceeds GitHub's file size limits and is therefore hosted externally.

Dataset Link:
https://drive.google.com/file/d/122LojSfMhzxY9rvQLIgK4tadz2wi2py1/view?usp=sharing

## Dataset Features

* Team 1
* Team 2
* Venue
* Toss Winner
* Toss Decision
* Match Winner
* Season

The datasets were cleaned and preprocessed before training the machine learning models.
